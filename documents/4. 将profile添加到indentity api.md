# 认证流程

1. 客户端 向 Identity Server 请求 token，使用 android client_id
2. Identity Server 验证客户端并颁发包含 gateway_api scope 的 token
3. 客户端 携带 token 访问 API Gateway
4. API Gateway 向 Identity Server 验证 token 的有效性和 scope
5. 验证通过后，API Gateway 将请求转发到后端服务
