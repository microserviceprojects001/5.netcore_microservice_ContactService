2. 你需要 RabbitMQ 服务器吗？
   是的，必须要有！ RabbitMQ 就像数据库一样，是一个独立运行的服务器。

类比理解：

- 数据库：SQL Server / MySQL / PostgreSQL

- 缓存：Redis

- 消息队列：RabbitMQ / Kafka

- 服务发现：Consul

# 5672 这个端口如何在代码中体现的呢

关键点：当你只设置 HostName 而没有设置 Port 时，RabbitMQ.Client 库会自动使用 5672 作为默认端口。
